Palindrome Checker

What is a Palindrome:

A palindrome is a word, phrase, number, or other sequence of characters that reads the same forward and backward (ignoring spaces, punctuation, and capitalization). For example, "A man, a plan, a canal, Panama" is a palindrome.


How to Use:

Enter a text in the input box below and click the "Check" button or press Enter to see if it's a palindrome.
