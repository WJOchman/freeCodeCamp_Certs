US Phone Number Validator

What is a US Phone Number Validator:

This application validates US phone numbers. A valid US phone number can be in various formats such as "1 555-555-5555", "(555) 555-5555", "555-555-5555", "5555555555", etc.


How to Use:

Enter a phone number in the input box below and click the "Check" button to validate the number. Use the "Clear" button to reset the input.
